# SublimeGtags

This is a plugin for the [Sublime Text 2](http://www.sublimetext.com/) text
editor that support [GNU GLOBAL(gtags)](http://www.gnu.org/software/global/)

## Installation
clone this repo directly into your Packages directory.

## Compatibility
 * OSX(good
 * Linux(less good
 * Windows(worst
